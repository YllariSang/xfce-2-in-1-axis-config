# What the hell is this?
---
EVOO 11.6" Convertible Touchscreen Laptop (2-in-1) is a cheap ass laptop that I got for free that is apparently from the mesopotamian era created by Walmart and now discontinued for some reason. This is just a custom script tailored around this laptop to make it rotate based on it's current orientation, so yeah rotate screen, mapping, and touch dynamically, because apparently the Linux kernel does not support it nor does any other intel driver and for some reason it works on windows (fucking gatekeepers). Anyways, this shit is super niche and really not applicable to anything else unless you install the dependencies such as iio-proxy-sensor and the likes, you can probably use it if you also have a 2-in-1 tablet laptop, just change the configuration based on what you have on god.

P.S. This is for XFCE, not sure if it would work on other DE's. But I think most likely will.
